# ima-notes 配置

配置文件默认路径：

```bash
skills/ima-notes/scripts/config.json
```

关键字段：

- `client_id`
- `api_key`
- `base_url`
- `request_timeout_seconds`

最小示例：

```json
{
  "client_id": "your_client_id",
  "api_key": "your_api_key",
  "base_url": "https://ima.qq.com",
  "request_timeout_seconds": 30
}
```
